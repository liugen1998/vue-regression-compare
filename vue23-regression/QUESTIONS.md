# QUESTIONS —— 需要业务/前端同学确认的事项

1. 登录方式与会话有效期？是否有验证码/扫码（决定 storage 还是 scripted，及录制频率）。
2. 20 个页面里是否存在 iframe 内嵌、或点击后**新开浏览器页签**的下钻？（当前版本仅支持同页签，需要的话提前告知以便扩展。）
3. 双端接口路径/参数是否完全一致？若不一致请提供对照表（填 apiMap）。
4. 响应里哪些字段注定不同（traceId、服务器时间、耗时统计、AB 实验标记…）？（填 apiIgnoreFields。）
5. 图表是否全部 ECharts？版本？生产构建是否暴露 `window.echarts`？（决定 L1/L2 路径与是否需要 relX/relY。）
6. 页面是否有轮播/跑马灯/实时刷新组件？（填 masks 或在测试环境关闭。）
7. 筛选组合中哪些是非法/无意义组合？（填 constraints，避免浪费用例。）
8. 性能达标口径确认：默认"中位数劣化不超过 10% 且不超过 100ms"，是否按此执行？
